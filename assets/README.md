# Assets Folder Structure

## /logos
- `logo.png` - Main logo with "Investment Bank" tagline (2040px, full color for light mode)
- `logo-white.png` - White version for dark mode (2040px)
- `favicon.ico` - Browser tab icon (ICO format)
- `favicon-16x16.png` - Small favicon
- `favicon-32x32.png` - Standard favicon
- `favicon-192x192.png` - Large favicon for modern browsers
- `apple-touch-icon.png` - iOS home screen icon (180x180)
- `favicon-source.png` - Original logomark used to generate favicons
